# MonkeyPatchers .github

This repository contains shared community health files for the
[MonkeyPatchers](https://github.com/MonkeyPatchers) organization.

Files in this repo are used as defaults for all repositories in the org
that don't provide their own versions.

## Included

- `SECURITY.md` – How to report security issues responsibly
- `CODE_OF_CONDUCT.md` – Expectations for behavior in the community
- `CONTRIBUTING.md` – Guidelines for contributing to MonkeyPatchers projects

These documents help keep our projects welcoming, secure, and maintainable.
